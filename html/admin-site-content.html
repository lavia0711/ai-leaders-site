사이트 콘텐츠 관리

히어로 배너, 강사진 소개, 출강 문의/강사 지원 폼의 선택 옵션을 Supabase 기준으로 수정합니다.

콘텐츠 데이터를 불러오는 중입니다.
배너 수정/추가
랜딩페이지 히어로 섹션에 반영됩니다.

이미지 1개만 등록하면 PC와 모바일에 자동 반응형으로 적용됩니다. 비워두면 기존 랜딩페이지 기본 이미지가 유지됩니다.

업로드된 비디오가 없으면 이미지 배너가 노출됩니다.

배너 목록
왼쪽 핸들을 드래그하면 공개 배너 순서가 자동 저장됩니다.
강사진 수정/추가
랜딩페이지 강사진과 소개페이지 이력 팝업에 반영됩니다.

이미지를 선택하면 원본 파일이 자동 업로드됩니다.

업로드된 원본 이미지 미리보기
강사진 목록
드래그로 순서를 저장하고, 숨김 버튼으로 공개 페이지 노출을 전환합니다.
드롭다운 옵션 수정/추가
출강 문의와 강사 지원 작성폼의 선택지를 관리합니다.
옵션 목록
선택한 그룹 안에서 드래그한 순서가 작성폼에 자동 반영됩니다.

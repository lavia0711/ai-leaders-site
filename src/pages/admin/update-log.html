  • 강의 일자와 신청자 수를 오름차순·내림차순으로 정렬할 수 있게 했습니다.
  • 신청 가능한 강연을 최대 8개까지 대표 강연으로 직접 지정하고 드래그 앤 드롭으로 노출 순서를 저장할 수 있게 했습니다.
  • 공개 홈페이지는 수동 지정 강연을 저장 순서대로 먼저 노출하고, 남은 자리는 기존 신청자 수와 강연 종료 시각 기준 자동 정렬로 채웁니다.

신청 관리 필터·중복 신청 판정

  • 신청 관리 목록과 엑셀 내보내기에 행사 일자를 추가하고, 신청일·행사 일자·강연·구분·신청자·연락처에 검색 가능한 복수선택 필터를 적용했습니다.
  • 필터 버튼의 크기와 배치를 고정하고, 한 번에 하나의 필터창만 열리도록 했으며 현재 필터 결과만 엑셀로 내보낼 수 있게 했습니다.
  • 강연명·신청자명·연락처가 모두 같은 경우 최초 1건 이후 신청을 중복 신청으로 자동 판정합니다.
  • 중복 신청은 별도 상태 배지와 복수선택 필터로 확인할 수 있고, 필터 결과 엑셀에도 정상·중복 상태가 포함됩니다.
  • 중복 신청은 데이터만 저장하고 신청자 수에는 반영하지 않으며, 삭제하거나 제외해도 신청자 수를 변경하지 않습니다.

출강 문의·강사 지원 목록 필터

  • 출강 문의에 접수일·기업/기관·담당자·연락처·진행 장소·지역·선호 강사·AI 수준·관심 주제 복수선택 필터를 추가했습니다.
  • 강사 지원에 접수일·지원자·연락처·활동 지역·강의 경력·강의 형태·전문 분야 복수선택 필터를 추가했습니다.
  • 각 필터는 선택지 검색·개별 초기화·전체 초기화·단일 메뉴 열림을 지원하며, 엑셀에는 현재 필터 결과만 포함됩니다.

강연 등록·수정 입력 개선

  • 무료·유료 강연을 선택 버튼으로 구분하고, 무료 강연에서는 가격·결제 계좌 입력을 숨기며 유료 강연에서만 관련 값을 입력하고 검증하도록 했습니다.
  • 유료 신청서의 기존 결제 관련 항목은 유지하면서 나이 항목을 추가하고, 신청 상태는 신청자 화면에 노출하지 않도록 했습니다.
